##hello,world

**Line #2:**第二行定义主函数

main() 函数是每个程序的入口点。 kotlin 中的所有函数都以 fun 关键字开头，后跟函数名称（这里 main 是函数名）、参数列表、可选返回类型和函数体（ { ……. } ）。 在这种情况下，主函数包含参数 - 字符串数组和返回单位。 Unit类型对应java中的void，表示该函数不返回任何值。

args：这是一个变量名，用于引用传递给 main 函数的命令行参数。

Array<String>：这是一个类型标注，表示 args 是一个 String 类型的数组。

Line #3: 第三行是一条语句，它打印“Hello, World!”打印程序的输出。

就像其他现代编程语言一样，Kotlin 中分号是可选的。

Kotlin Data Types

Kotlin 中最基本的数据类型是原始数据类型，所有其他数据类型都是引用类型，例如数组和字符串。Java 需要使用包装器 (java.lang.Integer) 来使原始数据类型表现得像对象，但 Kotlin 已经将所有数据类型都视为对象。

Kotlin 中有不同的数据类型：

  1. 整数数据类型 Integer Data type
  2. 浮点数据类型 Floating-point Data Type
  3. 布尔数据类型 Boolean Data Type
  4. 字符数据类型 Character Data Type

整数数据类型

这些数据类型包含整数值。

数据类型 位 最小值 最大值
byte 8位 -128 127
short 16位 -32768 32767
int 32位 -2147483648 2147483647
long 64位 -9223372036854775808 9223372036854775807

当初始化一个没有显式指定类型的变量时，编译器会自动推断为自 Int 起足以表示该值的最小类型。 如果不超过 Int 的表示范围，那么类型是 Int。 如果超过了，那么类型是 Long。 如需显式指定 Long 值，请给该值追加后缀 L。 显式指定类型会触发编译器检测该值是否超出指定类型的表示范围。

浮点数据类型

这些数据类型用于存储小数值或小数部分。

Data Type Bits Min Value Max Value
float 32 bits 1.40129846432481707e-45 3.40282346638528860e+38
double 64 bits 4.94065645841246544e-324 1.79769313486231570e+308

如需将数值转换为不同的类型，请使用显式转换。

###数字字面常量

数值常量字面值有以下几种:

  • 十进制:123
    • Long 类型用大写 L 标记: 123L
  • 十六进制: 0x0F
  • 二进制: 0b00001011

Kotlin 不支持八进制。

Kotlin 同样支持浮点数的常规表示方法:

  • 默认 double：123.5、123.5e10
  • Float 用 f 或者 F 标记: 123.5f

布尔数据类型

布尔数据类型仅表示true 或 false 的一位信息。Kotlin 中的布尔类型与 Java 中的布尔类型相同。这些运算析取 (||) 或合取 (&&) 可以对布尔类型执行。

boolean 1 bit true or false

字符数据类型

字符数据类型表示小写字母(az)、大写字母(AZ)、数字(0-9)和其他符号。

Data Type Bits Min Value Max Value
char 16 bits ‘\u0000’ (0) ‘\uFFFF’ (65535)

##Kotlin 变量

不可变变量 val

不可变也称为只读变量。因此，我们不能更改使用val关键字声明的变量的值。

注意：不可变变量不是常量，因为它可以用变量的值进行初始化。这意味着不可变变量的值不需要在编译时已知，并且如果在重复调用的构造中声明它，则它可以在每次函数调用时采用不同的值。
